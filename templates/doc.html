Documentation de l'API REST

Version 1 — Contraventions alimentaires Montréal

GET

/contrevenants

Retourne les établissements contrevenants entre deux dates.

Paramètres

Paramètre Type Requis Description
du string Oui Date de début (YYYY-MM-DD)
au string Oui Date de fin (YYYY-MM-DD)

Exemple

GET /contrevenants?du=2022-05-08&au=2024-05-15

Réponse 200

[
  {"etablissement": "RESTAURANT ABC", "nb_contraventions": 3}
]

Codes de statut

200
Liste des contrevenants avec leur nombre de contraventions.
400
Paramètres manquants ou date de début postérieure à la date de fin.
GET

/api/etablissements

Retourne tous les établissements ayant commis des infractions, triés par nombre d'infractions décroissant (JSON).

Réponse 200

[
  {"etablissement": "RESTAURANT ABC", "nb_infractions": 5},
  {"etablissement": "EPICERIE XYZ", "nb_infractions": 2}
]
GET

/api/etablissements.xml

Retourne les mêmes données que ci-dessus en format XML (UTF-8).

Réponse 200

<?xml version="1.0" encoding="UTF-8"?>
<etablissements>
  <etablissement>
    <nom>RESTAURANT ABC</nom>
    <nb_infractions>5</nb_infractions>
  </etablissement>
</etablissements>
GET

/api/etablissements.csv

Retourne les mêmes données en format CSV (UTF-8).

Réponse 200

etablissement,nb_infractions
RESTAURANT ABC,5
EPICERIE XYZ,2
GET

/api/etablissement/{nom}

Retourne toutes les infractions d'un établissement donné.

Codes de statut

200
Liste des infractions de l'établissement.
404
Établissement non trouvé.
POST

/api/inspections

Crée une demande d'inspection. Corps JSON requis et validé par json-schema.

Corps de la requête

{
  "etablissement": "string (requis)",
  "adresse": "string (requis)",
  "ville": "string (requis)",
  "date_visite": "YYYY-MM-DD (requis)",
  "prenom_client": "string (requis)",
  "nom_client": "string (requis)",
  "description": "string (requis)"
}

Codes de statut

201
Demande créée — retourne l'identifiant.
400
Données invalides ou champ manquant.
DELETE

/api/inspections/{id}

Supprime une demande d'inspection par son identifiant.

Codes de statut

200
Demande supprimée avec succès.
404
Demande non trouvée.
